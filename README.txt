Psychology Insights Magazine - GitHub Pages Template
---------------------------------------------------

This is a Jekyll-ready magazine-style theme scaffold created for GitHub Pages.

Quick setup:
1. Create a GitHub repository named: <yourname>.github.io (or any repo and enable GitHub Pages)
2. Upload the contents of this folder to the repository root.
3. Edit _config.yml: set `url`, `disqus.shortname` (replace 'your-disqus-shortname'), and `buttondown.url`.
4. Customize site title and author in _config.yml.
5. Add new posts to the `_posts/` folder using the YYYY-MM-DD-title.md format. Each post supports `featured_image` front matter.
6. If you use a custom domain, set `baseurl` accordingly.
7. Disqus: create a free Disqus shortname at https://disqus.com and update _config.yml and the embed script.
8. Buttondown: create your Buttondown newsletter and update the form action and _config.yml.

Files of interest:
 - index.html (homepage with featured grid)
 - _layouts/post.html (single post layout with Disqus embed)
 - assets/css/style.css (soft blue theme & dark mode toggle handled by JS in header)
 - _posts/2025-01-01-understanding-cognitive-biases.md (sample post)
